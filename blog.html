var : var can be declared multiple times, can assign new values and has a global scope.
let : let can only be declared once,can assign new value and it has block Scope (can't be used outside of a block of code).
const : const can only be declared once,can't assign new value and has block Scope (can't be used outside of a block of code).

Syntax : Arrow functions have simpler syntax than a regular function.
This keyword : "This" keyword gives different values in each depending on the situation.
Constructors/“new” keyword : We can easily construct objects with regular functions. However, arrow functions can not be used as constructors.
Arguments binding : In regular function, Arguments keywords can be used to access the arguments of which passed to function.Arrow functions do not have an arguments binding. However, if you want to access arguments in an arrow function, you can use the rest operator.

Map : It applies a function to each element of an array and returns the results in a new array.
For..Each : It applies a function to each element of an array and does not return any value.
Filter : It applies a condition to each element of the array and returns all the values that meets the condition in a new array.
Find : It applies a condition to each element of the array and returns only the first value that meets the condition and stops.

We can create strings that are complex and contains dynamic elements. Template string lets us insert variables and expressions into strings without the need of concatenate.
